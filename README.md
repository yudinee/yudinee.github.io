# yudinee Portfolio

Spring Boot 기반 백엔드 프로젝트를 중심으로  
기획 → 설계 → 구현 → 배포까지의 경험을 담은 포트폴리오입니다.

🔗 **Live Site**: https://yudinee.github.io  

---

## 🚀 CI / CD

GitHub Actions를 활용하여  
**정적 포트폴리오 사이트의 빌드 및 배포를 자동화**했습니다.

### 🔄 Deployment Flow

```text
GitHub Repository
        ↓
GitHub Actions (Build & Deploy)
        ↓
GitHub Pages (yudinee.github.io)
```

---

## 🛠 Tech Stack

- **Backend**: Java, Spring Boot, Spring MVC, JPA/Hibernate
- **Database**: MySQL
- **Frontend**: HTML5, CSS3, JavaScript, React
- **DevOps**: GitHub Actions, GitHub Pages
